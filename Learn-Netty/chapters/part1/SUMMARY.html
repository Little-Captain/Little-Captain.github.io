1. 第一部分 -- Netty 的概念及体系结构

Netty 是一款用于创建高性能网络应用程序的高级框架。

使用 Netty 构建应用程序, 开发者不必是一名网络编程专家。使用 Netty 比直接使用底层的 Java API 容易得多。Netty 推崇良好的设计实践, 如, 将应用程序逻辑和网络层解耦。
