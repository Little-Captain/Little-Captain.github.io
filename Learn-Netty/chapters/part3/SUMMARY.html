1. 第三部分 -- 网络协议

WebSocket 是一种为了提高 Web 应用程序的性能以及响应性而开发的先进的网络协议。

  • 第12章 WebSocket

    • 通过构建一个可以在多个浏览器客户端之间进行实时通信的聊天室, 将学习到如何使用 WebSocket 来实现双向数据传输
    • 还将会学到如何在应用程序中通过检测客户端是否支持 WebSocket 协议, 从而从 HTTP 协议切换到 WebSocket 协议

  • 第13章 使用 UDP 广播事件

    • 学习 Netty 对于用户数据报协议(UDP)的支持
    • 构建可适用于多种实际用途的广播服务器和监视器客户端
