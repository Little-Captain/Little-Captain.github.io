# 文件相关命令

## file(determine file type)

* 确定文件类型
    * 查看文件内容的简短说明
    * Linux 系统中的文件名不需要反应文件的内容

```bash
file filename
```

## less(opposite of more)

* 查看文本文件内容

* less 查看文件时，常用的键盘命令
    * vim 中的上下滚动命令均适用
    * /characters ：向前查找指定字符串
    * q ：退出 less

```bash
less filename
```
