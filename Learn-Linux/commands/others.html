1.1. exit

  • 关闭当前会话
exit

1.2. echo

display a line of text

  • 显示一行文本。把文本参数内容打印到标准输出

1.3. clear

clear the terminal screen

  • 清屏

1.4. history

GNU History Library

  • 显示历史列表的记录
history | less
history | grep text
