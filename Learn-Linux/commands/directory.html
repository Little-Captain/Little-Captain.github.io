1. 目录相关命令

1.1. pwd(print name of current/working directory)

  • 查看当前工作目录
pwd

1.2. cd(change directory)

  • 改变工作目录
cd directory # 切换到指定目录
cd # 将工作目录改变为主目录(家目录)
cd - # 将工作目录改变成先前的工作目录
cd ~ # 将工作目录改变为当前用户的主目录
cd ~username # 将工作目录改变为 username 的主目录

1.3. ls(list directory contents)

  • 列出目录内容
ls
ls directory
ls file
