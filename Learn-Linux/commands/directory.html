1. 目录相关命令

1.1. pwd

print name of current/working directory

  • 查看当前工作目录
pwd

1.2. cd

change directory

  • 改变工作目录
cd directory # 切换到指定目录
cd # 将工作目录改变为主目录(家目录)
cd - # 将工作目录改变成先前的工作目录
cd ~ # 将工作目录改变为当前用户的主目录
cd ~username # 将工作目录改变为 username 的主目录

1.3. ls

list directory contents

  • 列出目录内容

  • 可指定多个目录

ls ~ /usr
短选项 长选项 说明
-a --all 列出所有文件，包括以点开头的隐藏文件
-d --directory 一般和 -l 结合使用，用于查看目录的详细信息，而不是目录中的内容
-F --classify 在每个所列出的名字后面加上类型指示符。目录名，会加上一个斜杠
-h --human-readable 和 -l 结合使用。以可读的方式显示文件大小(默认以字节数显示文件大小)
-l - 将输出以长格式显示
-r --reverse 结果以倒序输出
-S - 按文件大小对结果排序，最大文件在最前面
-t - 按修改时间排序，最近修改在最前面
-i -inode 显示索引节点(inode)

1.3.1. 长格式含义

drwxrwxrwx 1 root root 4096 Feb 15 17:48 miniconda3

  • drwxrwxrwx
    • d/l/-
      • d 目录
      • l 符号链接/软链接/symlink
      • - 普通文件
    • rwxrwxrwx
      • 3位一组，分别代表[文件/目录]的[所有者/所属组/其他所有人]的访问权限
    • 1
      • 文件硬链接数目
    • root
      • 文件所有者的用户名
    • root
      • 文件所属用户组的名称
    • 4096
      • 以字节数表示的[文件/文件夹]的大小
    • Feb 15 17:48
      • 上次修改文件的日期和时间
    • miniconda3
      • [文件/文件夹]的名称
ls
ls directory
ls file
