1. 时间相关命令

1.1. date

print or set the system date and time

  • 查看时间和日期
date +"%Y-%m-%d(%A) %T"

1.2. cal

display a calendar

  • 查看日历
cal # 当月日历
cal 1991 # 查看 1991 年的日历

1.3. time

time functions for gawk

  • 用于量测特定指令执行时所需消耗的时间及系统资源等信息。
