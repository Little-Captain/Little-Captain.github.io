1. 定制 shell 提示符

1.1. 提示符分解

  • 默认提示符 [lc@study vim]$

    • lc 用户名
    • study 主机名
    • vim 当前工作目录

  • 提示符样式由名为 PS1 的环境变量定义

    • PS1(prompt string 1)
    • [\u@\h \W]\$

  • 定义 shell 提示符时使用的转义字符

  • 设计提示符

| 转义字符 | 含义 | | :-: | :-: | | \a | 提示符铃声 | | \A | 24 小时制时间，格式为小时:分钟；在提示符中添加提示符信息，可以确定某些任务的执行时间 | | \h | 主机名，不带域名 | | \[ 和 \] | 用于封装非打印字符串 | | ... | 参考上表 |

1.2. 添加颜色

  • 字符颜色是由发送到终端的一个 ANSI 转义码来控制的，该转义码要嵌入到需要显示的字符流中

  • 一个 ANSI 转义码以八进制 033 开始，后跟一个可选的字符属性，之后是一条指令

    • \033[0;30m：正常文本属性(0)，黑色

  • 文本颜色的转义序列

    • 文本属性
      • 0：正常，深色
      • 1：粗体，浅色
    • PS1="\[\033[0;31m\][\u@\h \W]\$\[\033[0m\] "
      • \[\033[0;31m\]：设置颜色
      • \[\033[0m\]：恢复到原来的颜色

  • 背景颜色的转义序列
    • 背景颜色不支持粗体属性
    • PS1="\[\033[0;41m\][\u@\h \W]\$\[\033[0m\] "
      • \[\033[0;41m\]：设置颜色
      • \[\033[0m\]：恢复到原来的颜色

  • 文本属性
    • 0：正常
    • 1：粗体
    • 4：下划线
    • 5：闪烁，很多终端不支持闪烁属性
    • 7：斜体

1.4. 保存提示符

  • .bashrc 文件中写入 PS1 的设置项
# 注意这里一定要使用单引号，抑制所有的 shell 扩展
PS1='Your Setting'
export PS1
  • export 命令
    • export 的基本作用就是将父 shell 中的局部变量设置为环境变量，使得该变量可以在子 shell 中使用
    • 变量是否使用 export 导出为环境变量
      • 使用：导出为环境变量，可供子 shell 使用
      • 不使用：仅当前 shell 可以使用，子 shell 不可使用
