1. 数组

1.1. 什么是数组

  • 数组是可以一次存放多个值的变量, 数组的组织形式如同表格一样。

  • 数组中的单元叫做元素, 并且每个元素中含有数据。使用一种叫做索引或是下标的地址就可以访问一个独立的数组元素。

  • bash 中的数组是一维的。

  • bash 从第二个版本开始提供对数据的支持。最初的 UNIX shell 程序 sh 是不支持数组的。

1.2. 创建一个数组

  • 命名数组变量同命名其他 bash 变量一样, 当访问数组变量时可以自动创建它们。

  • 示例

a[1]=foo
echo ${a[1]} # foo

  • 通过第一条命令, 可将值 foo 赋给数组 a 的元素 1。第二条命令显示了元素 1 的存储值。在第二条命令中使用花括号是为了阻止 shell 在数组元素名里试图扩展路径名。

  • 使用 declare 命令也可以创建数组。

# 这是使用选项 -a 和 declare 创建数组 a 的实例。
declare -a a

1.3. 数组赋值的方式有两种

1.3.1. 赋单一值

name[subscript]=value
  • 这里的 name 是数组名, 并且 subscript 是大于或等于 0 的整数(或算术表达式)。value 是赋给数组元素的字符串或整数。

1.3.2. 赋多值

name=(value1 value2...)

  • 这里的 name 是数组名, 并且将 vatue1 value2... 等值依次赋予从元素 0 开始的数组元素。

  • 示例

days=(Sun Mon Tue Wed Thu Fri Sat)
# 通过为每个值指定一个下标来给特定元素赋值也是可行的。
days=([0]=Sun [1]=Mon [2]=Tue [3]=Wed [4]=Thu [5]=Fri [6]=Sat)

1.4. 访问数组元素

  • 访问语法
${array[index]}

1.5. 数组操作

1.5.1. 输出数组的所有内容

  • 可以使用下标 "*" 和 "@" 来访问数组中的每个元素。

  • 对于定位参数来讲, 符号 "@" 较之更有用。

  • 示例

animals=("a dog" "a cat" "a fish")
for i in ${animals[*]}; do echo $i; done
# a
# dog
# a
# cat
# a
# fish
for i in ${animals[@]}; do echo $i; done
# a
# dog
# a
# cat
# a
# fish
for i in "${animals[*]}"; do echo $i; done
# a dog a cat a fish
for i in "${animals[@]}"; do echo $i; done
# a dog
# a cat
# a fish

  • 如果不对符号 ${array[*]} 和 ${array[@]} 加以双引号包裹, 它们会得到相同的结果，即使用空白字符分割所有内容后的结果。

  • 如果对符号 ${array[*]} 和 ${array[@]} 加以双引号包裹, 就会得到不同的结果。符号 "*" 将数组所有内容放在一个字中, 而符号 "@" 使用 3 个字来显示数组的真实内容。

1.5.2. 确定数组元索的数目

  • 可以采用类似获取字符串长度的方式来确定数组中元素的数目。

  • 示例

a[100]=foo
# 数组中元素的个数，未被赋值的位置不会被计数
echo ${#a[@]} # 1
# 100 号元素的字符串长度
echo ${#a[100]} # 3

1.5.3. 查找数组中使用的下标

  • 由于 bash 允许在下标赋值中包含 "空格", 有时这对确定实际存在的元素是很有用的。

  • 这可以过参数扩展来实现

${!array[*]}
${!array[@]}

  • 这里的 array 是数组变量名。就像符号 "*" 和 "@" 等参数扩展一样, 引用中含有的 "@" 形式是最有用的, 因为它将数组内容扩展成独立的单词。

  • 示例

foo=([2]=a [4]=b [6]=c)
for i in "${foo[@]}"; do echo $i; done
# a
# b
# c
for i in "${!foo[@]}"; do echo $i; done
# 2
# 4
# 6

1.5.4. 在数组的结尾增加元素

  • 通过使用 "+=" 赋值运算符, 可以在数组的尾部自动地添加元素。

  • 示例

foo=(a b c)
echo ${foo[@]} # a b c
foo+=(d e f)
echo ${foo[@]} # a b c d e f

1.5.5. 数组排序操作

  • shell 虽然没有直接的方式来完成排序功能, 但是用一些代码来完成并非难事。

  • 示例

a=(f e d c b a)
echo "Orginal array: ${a[@]}" # Orginal array: f e d c b a
a_sorted=($(for i in "${a[@]}"; do echo $i; done | sort))
# 使用外层括号，分隔符为空格，不使用为换行符
echo "Sorted array: ${a_sorted[@]}" # Sorted array: a b c d e f

1.5.6. 数组的删除(bash 中适用)

  • 使用 unset 命令, 可以删除数组。
foo=(a b c d e f)
echo ${foo[@]}
unset foo
echo ${foo[@]}
  • 也可以使用 unset 来删除单个数组元素。
foo=(a b c d e f)
echo ${foo[@]}
unset 'foo[2]' # 这里要使用单引号来阻止扩展
echo ${foo[@]}
  • 对数组元素赋一个空值并不意味着清空它的内容。
foo=(a b c d e f)
foo=
echo ${foo[@]} # b c d e f
# 结果原因见下一条
  • 任何涉及到不含下标的数组变量的引用指的都是数组中的元素 0。
foo=(a b c d e f)
echo ${foo[@]} # a b c d e f
foo=A
echo ${foo[@]} # A b c d e f
