1. 异步执行

  • Linux 中的 shell 脚本是可以在多任务中运行的。这涉及到父脚本以及一个或多个子脚本的加载问题, 子脚本可以在父脚本运行时执行其他额外的任务。

  • 当一系列脚本以这种方式运行的时候, 保持父脚本与子脚本的协调一致就会是一个问题。

  • 也就是说, 如果父脚本与子脚本彼此相互依赖, 一个脚本必须等待另一个脚本任务完成之后才能继续完成自己的任务。

  • bash 提供了一个内置的命令来帮助管理异步执行。wait 命令可以让父脚本暂停, 直到指定的进程(如子脚本)执行结束。
